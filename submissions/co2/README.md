# Bài nộp CO2

Bài toán: dự đoán lượng phát thải CO2 của xe bằng các thông số kỹ thuật và tiêu thụ nhiên liệu.

Nội dung thư mục:
- co2.ipynb: notebook nộp bài
- data/FuelConsumptionCo2.csv: bộ dữ liệu gốc
- figures: biểu đồ EDA và đánh giá mô hình
- tables: bảng metric, feature importance, kết quả cross-validation
- requirements.txt: danh sách thư viện cơ bản

Nguồn dữ liệu:
- IBM Skills Network course data
- Dữ liệu gốc liên quan đến fuel consumption và CO2 emissions tại Canada
